О программе

Програма «Аудит без границ» была создана группой разработчиков из Украины.

Она не имеет аналогов на просторах СНГ, т.к. учитывает требования законодательства, и может быть изменена в соответствии с последними изменениями в налоговом кодексе.

Програма «Аудит без границ» создавалась после тщательного анализа современной налоговой ситуации в постсоветском пространстве.
